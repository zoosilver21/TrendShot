🔑 YouTube Data API v3 키 설정

Google Cloud Console에서 발급받은 API 키를 입력해 주세요. 키는 브라우저 로컬에 안전하게 저장됩니다.

API 키를 입력하고 저장해 주세요.
💡 API 키 발급 방법:
1. Google Cloud Console 접속 → 새 프로젝트 생성
2. API 및 서비스 → 라이브러리 → YouTube Data API v3 사용 설정
3. 사용자 인증 정보 → API 키 생성 → 복사 후 위에 입력
4. 일일 무료 할당량: 10,000 유닛 (검색 1회 = 100유닛)
